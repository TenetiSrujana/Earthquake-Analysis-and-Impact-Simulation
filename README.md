# Earthquake-Analysis-and-Impact-Simulation
Quakes drop, we pop! Explore 30 days of real earthquakes, check out cool maps, and dive into a fun “What If” game where you get to test quake powers and watch the impact happen!

## Dataset
USGS Earthquake Data (last 30 days)

## Technologies Used
Python
Pandas
Matplotlib
Plotly
Jupyter Notebook

## How to Run
1. Download the repository
2. Open Earthquake_Analysis.ipynb in Jupyter Notebook
3. Run all cells

## Visualizations
- Earthquake magnitude distribution
- Location heatmaps
- Impact simulation scenarios
